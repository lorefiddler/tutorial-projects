Headings

Headings define the outline of your site. There are six levels of headings.

Lists

This is how you make an unordered list:

  • Add a "ul" element (it stands for unordered list)
  • Add each item in its own "li" (list item) element
  • They don't need to be in any particular order

  • You can add a paragraph to a list, as long as it's nested within list tags

This is what an ordered list looks like:

  1. Ordered lists are wrapped with an "ol" element
  2. The "li" element is the same
  3. The numbers increment on their own
